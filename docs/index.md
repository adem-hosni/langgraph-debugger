# LangGraph Debugger — Documentation

Welcome to the LangGraph Debugger project documentation. This folder
provides an overview of the repository, architecture, developer setup,
and usage instructions for both the server and client components.

Files
- `architecture.md` — high-level system architecture and component
  responsibilities.
- `setup.md` — environment setup and how to run the app locally.
- `usage.md` — runtime examples and common workflows.
- `development.md` — notes for contributors and how the codebase is
  structured.
- `contributing.md` — contribution guidelines and PR checklist.

Use the sidebar or open individual files to get started.
